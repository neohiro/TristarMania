# Tristar Mania
An addictive python game (Linux/Windows)

- GREEN TRIANGLE = NEUTRAL SHIP
- RED SQUARE = ENEMY VESSEL
- PINK ARROW = YOUR VESSEL
- GREY CIRCLE = ASTEROIDS TO MINE FOR REPAIRS
